# SQUARED project based on redux-saga example

## Install

```bash
git clone https://github.com/MartianLee/react-nextjs.git
yarn
yarn dev
```

## Use

Open ``localhost:3000`` in web browser 

Run Storybook with ``npm run storybook``

## Libraries

* React
* Redux
* Redux-saga
* Storybook
